## Telegram eBay Checker Bot

Telegram-бот для проверки товаров на eBay по ID товара с системой баланса, ротацией прокси и многошаговым взаимодействием с пользователем.

## Возможности

- Проверка товара по ID eBay
- Парсинг названия, цены и наличия товара
- Система баланса пользователя
- Подтверждение списания перед проверкой
- Имитация пополнения баланса
- Ротация прокси для запросов
- Автоматическое удаление нерабочих прокси во время текущего сеанса
- Многошаговые диалоги на основе FSM
- Интерфейс с inline- и reply-кнопками
- Поддержка SQLite

## Технологии

- Python 3.11+
- aiogram 3.x
- Requests
- Beautiful Soup
- SQLite
- aiosqlite

## Установка

1. Клонирование репозитория

```bash
git clone https://github.com/YOUR_USERNAME/telegram-ebay-checker.git
cd telegram-ebay-checker
```

2. Создание виртуального окружения

```bash
python -m venv venv
```
3. Активация виртуального окружения

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

4. Установка зависимостей

```bash
pip install -r requirements.txt
```

## Конфигурация

`config.py`

```python
import os
from dotenv import load_dotenv

load_dotenv()

BOT_TOKEN = os.getenv("BOT_TOKEN")

CHECK_PRICE = 0.05
MIN_DEPOSIT = 2

PROXY_FILE = "proxies.txt"
DATABASE_PATH = "database/database.db"
```

## Логика работы бота

## Проверка товара

1. Пользователь нажимает «Проверить товар»
2. Пользователь отправляет ID товара eBay
3. Бот показывает стоимость проверки
4. Пользователь подтверждает действие через inline-кнопку
5. С баланса списываются средства
6. Бот парсит страницу товара через прокси
7. Бот возвращает полученные данные
8. 
## Какие данные парсятся

- Название товара
- Цена товара
- Статус наличия

## Система прокси

Прокси хранятся в:

```text
proxies.txt
```

Пример:

```text
http://127.0.0.1:8080
http://127.0.0.1:8081
```

Логика работы:

- Случайный выбор прокси
- Нерабочие прокси удаляются из очереди текущего запуска
- Исходный файл с прокси не изменяется

## Система баланса

Возможности:

- Имитация пополнения баланса
